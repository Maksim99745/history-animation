# История анимация

Исторический слайдер с временными отрезками и событиями.

## Технологии

- React 18
- TypeScript
- styled-components
- Webpack
- Swiper
- GSAP

## Установка

```bash
npm install
```

## Запуск проекта

```bash
npm run dev
```

Проект будет доступен по адресу http://localhost:3000

## Сборка

```bash
npm run build
```

## Проверка типов

```bash
npm run type-check
```
